# Assembler and Disassembler

## Getting Started

`git clone git@github.com:dst2609/CMPE-220.git`

`cd CMPE-220/api`

`npm install`

`npm start` [Backend is up and running]

--Open new terminal tab

`cd CMPE-220/ui`

`npm install`

`npm run dev` [Frontend is up and running at http://localhost:5173/]
